# 方法论
> 让我们能优秀的完成任务的一系列方法。

整体
* PDCA 循环。 来优化做任务的方法。 还有 OODA循环，与这个类似。
* 时间管理
  * 重要与紧急。重要且紧急的马上做；重要不紧急的，制定计划做；紧急不重要：请别人代劳；不紧急不重要：不做。
  * 二八原则。 重视二，忽略八。

1 开始任务前，获取信息
* 快速阅读
* 做笔记
* 学习 - 刻意练习

2 决策
* [SWOT](swot)
* PNI: 正面，负面，有趣面

3 计划
* SMART

4 执行
* 分析问题
  * [5W1H分析法](5W1H分析法)

## 其他方法
* [Scrum](scrum) 敏捷开发。
